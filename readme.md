### ML Project

This project aim to recognize letter from an image

For python version, you may run :

```console
$ python --version
```

which should return

```console
$ python --version
Python 3.6.7
```

or higher.

Don't forget to satisfied the requirements

```console
$ pip install -r requirements.txt
```

## How to

Launch it like this :

```console
$ cd transform
$ python api.py
```
